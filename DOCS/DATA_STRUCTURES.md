# Data Structure Implementation Guide

## Overview
This guide documents the custom data structure implementations for the Flight Management System.

## Implementations

### 1. Binary Search Tree
- File: `FlightBinarySearchTree.cs`
- Time Complexity: O(log n) average, O(n) worst
- Use Cases: General purpose flight searches

### 2. AVL Tree  
- File: `FlightAVLTree.cs`
- Time Complexity: O(log n) guaranteed
- Use Cases: Production systems requiring consistent performance

### 3. Hash Table (Planned)
- Use Cases: Quick lookups by ID
